# 청음록 Product Direction

## 한 문장 정의

**취향을 넓히며, 나만의 울타리를 조금씩 넓혀가는 음악 커뮤니티.**

Letterboxd의 기록성 + 왓챠피디아의 평점 UX + Pitchfork/IZM의 음악 비평 감성을 바탕으로, 익숙한 음악 너머의 좋은 노래와 음반을 함께 발견하고 평가하고 이야기하는 서비스.

## 핵심 레퍼런스

### 1. Letterboxd
- 개인 감상 기록
- 별점/리뷰/피드
- 프로필 아카이브
- 리스트 만들기
- 팔로우 기반 취향 발견

청음록 적용:
- 내 청음 기록
- 앨범/곡 리뷰 피드
- 팔로우 피드
- 올해의 음악 기록
- 명반/테마 리스트

### 2. 왓챠피디아
- 한국어 사용자에게 익숙한 별점 UX
- 취향 분석
- 추천 흐름
- 작품 상세 정보 구조

청음록 적용:
- 0.5 단위 별점
- 평균 별점/내 별점 비교
- 취향 태그
- 장르/무드 기반 추천

### 3. Pitchfork
- 음악 뉴스/비평 매체 감성
- 에디터 리뷰
- 점수/헤드라인 구조
- 신보/씬 뉴스

청음록 적용:
- 최근 음악 뉴스
- 에디터 리뷰
- 장문 평론
- 주간 추천 앨범

## MVP 우선순위

1. 검색 → 앨범 상세 → 리뷰 작성 흐름
2. 별점 입력 UX
3. 리뷰 피드
4. 내 프로필/청음 기록
5. 최근 뉴스/에디터 섹션
6. Supabase 로그인/DB
7. Spotify API 검색

## UI 방향

- 따뜻한 종이/아카이브 톤
- 카드형 앨범 UI
- 읽기 좋은 리뷰 피드
- 음악 매거진 같은 뉴스 카드
- 모바일에서 기록 작성이 편한 구조
